# PromoTimeCode

**Реферальная программа для YouTube-блогеров на вашем сервере Minecraft**

---

## 📖 Описание

PromoTimeCode — это мощный плагин для серверов Minecraft, который создает реферальную программу для YouTube-блогеров и стримеров.

Ютуберы создают уникальные промокоды, игроки их активируют, отыгрывают определенное время на сервере и получают донат (группу через LuckPerms), а ютубер получает денежное вознаграждение за каждого приведенного игрока.

---

## ✨ Возможности

- 🎯 Создание уникальных промокодов
- ⏱️ Автоматический подсчёт времени (только в онлайне)
- 🎁 Автовыдача групп через LuckPerms
- 💰 Баланс ютубера
- 🛡️ Защита от накруток (IP-лимит, AFK-проверка)
- ⚙️ Гибкая настройка сообщений
- 📊 Экспорт данных в CSV
- 💾 Система бэкапов
- 🌐 Мультиязычность (ru, en)

---

## 📋 Команды

### Для всех игроков
| Команда | Описание |
|---------|----------|
| `/promo help` | Показать справку |
| `/promo <код>` | Активировать промокод |
| `/promo progress` | Показать прогресс |

### Для ютуберов
| Команда | Описание |
|---------|----------|
| `/promo create <код>` | Создать промокод |
| `/promo info` | Информация о вашем коде |

### Для администраторов
| Команда | Описание |
|---------|----------|
| `/promo show <код>` | Информация о коде |
| `/promo balance add/remove <код> <сумма>` | Изменить баланс |
| `/promo delete <код>` | Удалить промокод |
| `/promo reload` | Перезагрузить плагин |
| `/promo ipstats <игрок>` | Статистика по IP |
| `/promo top` | ТОП промокодов |
| `/promo export` | Экспорт данных в CSV |
| `/promo backup create/list/restore` | Управление бэкапами |

## ⚙️ Установка

1. Установи **LuckPerms** на сервер
2. Скачай `PromoTimeCode.jar` из [релизов](https://github.com/Henezi/PromoTimeCode/releases)
3. Скопируй в папку `plugins/`
4. Перезапусти сервер
5. Настрой конфиги под себя
